## 40 PIN GPIO

### GPIO voltage

| GPIO       | Voltage | Max Voltage |
| ---------- | ------- | ----------- |
| All GPIO   | 3.3V    | 3.63V       |
| SARADC_IN4 | 1.8V    | 1.98V       |

### GPIO interface

Radxa NX5 IO provides a 40pin GPIO socket, which is compatible with most sensor applications on the market.
